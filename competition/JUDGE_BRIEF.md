# Judge brief

PROMETHEUS moves AI engineering from output generation to evidence-governed capability accumulation.

| Organ | Demonstrated behavior |
|---|---|
| SECA | Blocks promotion after a real replay-safety failure |
| HYDRA | Applies a behavior-changing idempotency repair |
| ProofGrid | Hashes artifacts and verifies the final receipt |
| Capability Genome | Encodes the repair, applicability, and exclusions |
| Build Truth | Authorizes promotion only after evidence verifies |

The second task is related but non-identical: it publishes a release manifest rather than a proof receipt. It reuses the same verified failure-class strategy proactively.

Measured result: **1 failed action → 0 failed actions**.
